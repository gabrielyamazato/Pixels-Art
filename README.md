
# Pixels Art

Meu primeiro projeto desafiador na Trybe. O objetivo do projeto é criar um board de pixels onde a pessoa usuaria pode pintar.




## Aprendizados

Com esse projeto aprendi sobre DOM e como manipular ele, o maior desafio foi desenvolver uma logica para trocar a cor selecionada.


## Screenshots

![App Screenshot](https://cdn.discordapp.com/attachments/1098241441580449892/1098241486677614684/PIXELS_ART.gif)


## Funcionalidades

- Paleta de cores padrão
- Botão de aleatorio para as três ultimas cores
- Botão de clear que limpa o board


## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:gabrielyamazato/Pixels-Art.git
```

Entre no diretório do projeto

```bash
  cd Pixels-Art
```

Abra o VSCode

```bash
  code .
```

Instale a extensão Live Server e clique em Go Live
